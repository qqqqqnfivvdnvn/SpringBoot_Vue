# 主数据项目实现说明

## 项目概述
主数据项目（MainData）是一个基于高德地图 API 的地理位置数据管理系统，支持批量上传地址数据并自动获取经纬度坐标信息。

---

## 技术栈

### 后端
- **框架**: Spring Boot 3.4.1 + Java 21
- **ORM**: MyBatis Plus
- **数据库**: SQL Server (haosen_project 库)
- **Excel 处理**: EasyExcel 3.3.2
- **HTTP 客户端**: Apache HttpClient

### 前端
- **框架**: Vue 3
- **UI 组件**: Element Plus
- **路由**: Vue Router
- **HTTP**: Axios

---

## API 接口说明

### 批次管理

#### 1. 获取批次列表
```
GET /api/maindata/batch/getBatchList
参数：pageNum, pageSize, batchId, status, startTime, endTime
```

#### 2. 上传批次文件
```
POST /api/maindata/batch/uploadFile
参数：file (Excel 文件)
```

#### 3. 导出批次数据
```
GET /api/maindata/batch/exportBatch
参数：batchId
```

### 地理位置数据

#### 1. 获取数据列表
```
GET /api/maindata/location/getDataList
参数：pageNum, pageSize, batchId, originalName, province, city, areaName
```

#### 2. 导出数据
```
GET /api/maindata/location/exportData
参数：batchId, originalName, province, city, areaName
```

## 核心功能流程

### 文件上传处理流程

1. **前端上传文件** → 后端保存文件到服务器
2. **创建批次记录** → 生成 UUID 作为批次 ID，状态设为"处理中"
3. **异步处理数据**:
   - 验证 Excel 表头（必需字段：id、省份、名称、地址）
   - 解析 Excel 数据
   - 调用高德地图 API 获取地理位置信息
   - 批量插入到 md_location 表
   - 更新批次状态为"已处理"

### 高德地图 API 调用

- **API 地址**: `https://restapi.amap.com/v3/geocode/geo`
- **返回格式**: JSON
```json
{
  "infocode": "10000",
  "info": "OK",
  "location": "105.857014,32.427191",
  "province": "四川省",
  "city": "广元市",
  "area": "利州区",
  "areaid": "510802",
  "address": "广元市利州区苴国路 490 号"
}
```

- **多 key 轮换**: 配置了 4 个 API key，自动轮换使用

## 使用说明

### 1. 数据库初始化
执行 `src/main/resources/db/maindata_schema.sql` 创建数据库表

### 2. 配置数据库连接
在 `application.properties` 中配置 SQL Server 连接：
```properties
spring.datasource.dynamic.datasource.master_sqlserver.url=jdbc:sqlserver://localhost:1433;databaseName=test
spring.datasource.dynamic.datasource.master_sqlserver.username=sa
spring.datasource.dynamic.datasource.master_sqlserver.password=your_password
```

### 3. 启动项目
- 后端：运行 Spring Boot 应用
- 前端：`cd vue_demo && npm run serve`

### 4. 访问系统
1. 登录系统
2. 在项目管理页面点击"主数据管理"项目
3. 进入左侧菜单"经纬度获取"→"文件上传"→上传 Excel 文件
4. 等待后台处理完成后，在"批次列表"或"汇总数据"中查看结果

上传的 Excel 文件必须包含以下字段（第一行表头）：

| id | 省份 | 名称 | 地址 |
|----|------|------|------|
| 123 | 四川省 | 某某公司 | 成都市某某街道 |

## 注意事项

1. **高德 API Key**: 如需更换或添加 API key，请修改 `GaoDeMapUtil.java` 中的 `API_KEYS` 数组
2. **批量处理**: 每批最多插入 50 条数据，避免 SQL Server 参数超限
3. **文件上传**: 支持 .xlsx 和 .xls 格式，最大 10MB
4. **异步处理**: 文件上传后立即返回，后台异步处理，可通过批次列表查看处理状态
