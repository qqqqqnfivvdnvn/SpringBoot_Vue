## 1. 概述

### 1.1 目标
将登录/注册页面的设计风格与后台管理系统主界面统一，采用深色高级感配色、卡片式布局和微交互动画，提升整体专业度和设计感。

### 1.2 设计原则
- **一致性**：与后台系统使用相同的配色方案和组件风格
- **专业感**：采用现代流行的设计元素（卡片阴影、渐变背景、微动画）
- **可用性强**：保持清晰的视觉层次和流畅的交互体验
- **响应式**：支持移动端和桌面端自适应

---

## 2. 设计方案

### 2.1 布局结构

保留左右分栏布局，优化视觉层次：

```
┌────────────────────────────────────────────────────────┐
│                                                        │
│  ┌─────────────┐     ┌───────────────────────────┐    │
│  │             │     │    欢迎回来               │    │
│  │   🔒        │     │    请登录您的账户继续使用  │    │
│  │             │     │                           │    │
│  │  主数据      │     │    用户名  [________]     │    │
│  │  管理系统    │     │    密码    [________]     │    │
│  │             │     │                           │    │
│  │  统一管理    │     │    [   登录按钮   ]       │    │
│  │  高效协作    │     │                           │    │
│  │             │     │    新用户？注册            │    │
│  └─────────────┘     └───────────────────────────┘    │
│   左侧欢迎区          右侧表单区（带卡片容器）          │
│   蓝色渐变背景         浅灰背景 + 白色卡片             │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### 2.2 配色方案

| 元素 | 亮色模式 | 暗色模式 |
|------|---------|---------|
| 左侧背景 | `linear-gradient(145deg, #3b82f6 0%, #1d4ed8 100%)` | `linear-gradient(145deg, #0f172a 0%, #1e293b 100%)` |
| 右侧背景 | `#f8fafc` | `#0f172a` |
| 卡片背景 | `#ffffff` | `#1e293b` |
| 主按钮 | `#3b82f6` | `#3b82f6` |
| 主按钮 hover | `#2563eb` | `#60a5fa` |
| 输入框边框 | `#e2e8f0` | `#475569` |
| 输入框 focus | `#3b82f6` + 光晕 | `#60a5fa` + 光晕 |
| 标题文字 | `#1e293b` | `#f1f5f9` |
| 次要文字 | `#64748b` | `#94a3b8` |
| 链接颜色 | `#3b82f6` | `#60a5fa` |

### 2.3 组件样式

#### 卡片容器
```css
.form-card {
  background: #ffffff;
  border-radius: 16px;
  padding: 40px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.1);
}

html.dark .form-card {
  background: #1e293b;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.4);
}
```

#### 主按钮
```css
.submit-btn {
  background: #3b82f6;
  border-radius: 10px;
  padding: 12px 24px;
  font-weight: 600;
  transition: all 0.2s ease;
  box-shadow: 0 2px 8px rgba(59, 130, 246, 0.3);
}

.submit-btn:hover {
  background: #2563eb;
  transform: translateY(-2px);
  box-shadow: 0 4px 16px rgba(59, 130, 246, 0.4);
}
```

#### 输入框
```css
.form-input {
  border: 1.5px solid #e2e8f0;
  border-radius: 10px;
  padding: 11px 14px;
  transition: all 0.25s ease;
}

.form-input:focus {
  border-color: #3b82f6;
  box-shadow: 0 0 0 3px rgba(59, 130, 246, 0.15);
  outline: none;
}

html.dark .form-input {
  background: #0f172a;
  border-color: #475569;
  color: #f1f5f9;
}

html.dark .form-input:focus {
  border-color: #60a5fa;
  box-shadow: 0 0 0 3px rgba(96, 165, 250, 0.25);
}
```

### 2.4 微交互动画

| 交互 | 效果 |
|------|------|
| 输入框 focus | 边框变蓝 + 光晕扩散 |
| 按钮 hover | 上移 2px + 阴影加深 |
| 按钮 click | 上移效果取消 + 阴影变浅 |
| 链接 hover | 颜色加深 + 下划线 |
| Toast 出现 | 从上方滑入 + 弹性缩放 |

## 4. 响应式支持

### 断点：768px

**桌面端（> 768px）：**
- 左右分栏布局
- 左侧欢迎区占 50%
- 右侧表单区占 50%

**移动端（≤ 768px）：**
- 上下堆叠布局
- 顶部欢迎区高度 200px
- 底部表单区自适应
- 隐藏特性标签

---

## 5. 暗色模式支持

暗色模式通过 `html.dark` 类切换：

| 元素 | 亮色 → 暗色变化 |
|------|----------------|
| 左侧背景 | 蓝色渐变 → 深蓝黑渐变 |
| 右侧背景 | 浅灰 `#f8fafc` → 深色 `#0f172a` |
| 卡片 | 白色 → 深灰蓝 `#1e293b` |
| 文字 | 深色 → 浅色 |
| 输入框 | 白色背景 → 深色背景 |
| 按钮 | 亮蓝渐变 → 深蓝渐变 |
